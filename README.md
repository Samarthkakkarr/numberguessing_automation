<h2>📌NUMBER GUESSING AUTOMATION </h2>

<h3>1. Project Overview</h3>
<p>
The <b>Number Guessing Game Automation</b> is a simple and effective demonstration of Robotic Process Automation developed using UiPath Studio.
</p>

<h3>2. User Interaction</h3>
<p>
It shows how a software robot can interact with the user, take input, and process it efficiently.
</p>

<h3>3. Logical Processing</h3>
<p>
The automation performs a sequence of logical and mathematical operations to produce a consistent result.
</p>

<h3>4. Activities Used</h3>
<p>
It uses key UiPath activities such as <b>Assign, Input Dialog, Message Box, and Invoke Method</b>.
</p>

<h3>5. Interactive Features</h3>
<p>
The integration of <b>text-to-speech functionality</b> makes the interaction more engaging and interactive.
</p>

<h3>6. Learning Outcome</h3>
<p>
It is a <b>beginner-friendly project</b> that helps in understanding core RPA concepts and workflow design.
</p>

<h3>7. Practical Application</h3>
<p>
It demonstrates how automation can be applied to <b>structured and rule-based tasks efficiently</b>.
</p>
